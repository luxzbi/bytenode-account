연동 방법 1 — 간단 SSO 리다이렉트 (앱 등록 불필요)
로그인/가입 버튼을 아래 주소로 연결하면, 완료 후 redirect 주소로 #bn_token=<토큰>을 붙여 돌아갑니다. 
로그인:   https://bytenode-account.vercel.app/login?redirect=<내 서비스 주소>
회원가입: https://bytenode-account.vercel.app/welcome?redirect=<내 서비스 주소>
돌아온 페이지에서: 
const m = location.hash.match(/bn_token=([^&]+)/);
if (m) {
  localStorage.setItem('bn_token', decodeURIComponent(m[1]));
  history.replaceState(null, '', location.pathname);
}
연동 방법 2 — OAuth 2.0 authorization code (등록한 앱)
단계요청
1. 인가 요청GET /authorize?response_type=code&client_id=…&redirect_uri=…&state=…
2. 코드 수신로그인 완료 후 redirect_uri?code=…&state=…로 이동됨
3. 토큰 교환POST /token — { grant_type:"authorization_code", code, client_id, client_secret }
4. 유저 조회GET /userinfo — 헤더 Authorization: Bearer <access_token>
주의: 인가 코드는 2분 내 1회만 교환 가능. client_secret은 서버에서만 사용하고 절대 프론트엔드에 넣지 마세요. 토큰 검증은 항상 /userinfo로 하세요.

PKCE (S256) 지원: 서버 없이 브라우저/앱만으로 연동하려면 secret 대신 PKCE를 쓰세요. /authorize에 code_challenge(=BASE64URL(SHA256(code_verifier)))와 code_challenge_method=S256을 추가하고, /token 교환 시 code_verifier를 보내면 client_secret 없이 교환됩니다.
